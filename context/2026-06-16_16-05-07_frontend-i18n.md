# 2026-06-16 Frontend i18n (RU/EN/ZH)

## Что добавлено

- i18n на фронтенде через `i18next` + `react-i18next`
- Поддержка языков: `ru`, `en`, `zh`
- Переключатель языка в шапке (`Layout.tsx`)
- Переводы ключевых экранов и блоков:
  - Home (заголовок, CTA, meta котика, недавно сгенерированные)
  - Profile (заголовок, счётчик, состояния empty/loading)
  - Login/Register (заголовки, подписи полей, кнопки, переключатели)
- Перевод `pose` для `SeaLionCanvas` (используется в `aria-label`)

## Файлы

- `frontend/src/i18n/i18n.ts`
- `frontend/src/i18n/index.ts`
- `frontend/src/i18n/locales/{ru,en,zh}.json`

- `frontend/src/pages/ProfilePage.tsx`
- `frontend/src/pages/LoginPage.tsx`
- `frontend/src/pages/RegisterPage.tsx`
- `frontend/src/components/SeaLionCanvas.tsx`
