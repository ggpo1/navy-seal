# Бейджи достижений

## Все бейджи (12)

| ID | Название | Условие | Профиль | Карточка |
|----|----------|---------|---------|----------|
| `first_seal` | Первый котик | ≥1 котик | ✓ | только первый котик |
| `seals_5` | 5 котиков | ≥5 котиков | ✓ | все котики автора |
| `seals_25` | 25 котиков | ≥25 котиков | ✓ | все котики автора |
| `first_legendary` | Первый легендарный | ≥1 legendary | ✓ | все котики автора |
| `legendary_10` | 10 легендарных | ≥10 legendary | ✓ | все котики автора |
| `comments_100` | 100 комментариев | ≥100 комментов получено | ✓ | все котики автора |
| `ratings_50` | 50 оценок | ≥50 оценок получено | ✓ | все котики автора |
| `commenter_25` | Комментатор | ≥25 комментов оставлено | ✓ | все котики автора |
| `critic_50` | Критик | ≥50 оценок поставлено | ✓ | все котики автора |
| `popular_seal` | Популярный котик | 20+ оценок или 4.5+ при 10+ | ✓ если есть | только популярный котик |
| `top_week` | Топ недели | в топ-50 за 7 дней | ✓ если есть | только котик из топа |
| `hall_of_fame` | Зал славы | в топ-50 all-time | ✓ если есть | только котик из топа |

## Правила топов (`BadgeRules`)
- Топ: top 50, min 1 оценка, сортировка как в `GetTopAsync`
- Популярность: `RatingCount >= 20` OR (`RatingCount >= 10` AND `Average >= 4.5`)
